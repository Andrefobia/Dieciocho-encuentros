Para Sophi

I Para cuando estés triste. II Para cuando dudes de ti. III Para cuando sientas que Dios está lejos. IV Para cuando te sientas sola. V Para cuando fracases. VI Para cuando el mundo te haga sentir que no eres suficiente. VII Para cuando el miedo no te deje avanzar. VIII Para cuando necesites recordar que eres amada. IX Para cuando hayas pecado. X Para cuando reces y no sepas qué decir. XI Para cuando tengas una decisión difícil. XII Para cuando estés enferma. XIII Para cuando cumplas uno de tus sueños. XIV Para cuando te sientas incomprendida. XV Para cuando necesites volver a empezar. XVI Para cuando tengas que perdonar a alguien. XVII Para cuando pierdas a alguien. XVIII Para cuando quieras recordar quién eres.
